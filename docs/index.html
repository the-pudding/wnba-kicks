About this project

All shoes were drawn as vectors in Adobe Illustrator from reference photos found online. Some photos were of better quality than others, resulting in more detail on the shoes. The shapes were exported as SVGs and morphed using Flubber, a javascript library from Noah Veltman.

If you liked this project, check out our original morphing Air Jordans piece. You can buy this WNBA Kicks project and the Air Jordans version in poster form.

This project is not affiliated with any shoe brands featured including Adidas, Fila, Nike, Puma, or Reebok.
